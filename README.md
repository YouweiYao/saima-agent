# 赛马智能体

需求拆分工具

## 使用方法

```bash
# 主方案：按字符分片（500字符/块）
python3 saima_main.py <word文件>

# 备用方案：按句子分片（解决截断问题）
python3 saima_main.py <word文件> --sentence
```

## 问题记录

### 2026-03-02: 分片截断问题

**问题**：长段落按字符分片时被截断，LLM误解需求

**现象**：
- 原文："大模型知识库应具有但不限于以下功能："
- 被拆分到不同chunk
- LLM补全了原文中不存在的需求

**原因**：按固定字符数分片，可能在句子中间截断

**解决**：
1. 主方案：字符分片（简单快速）
2. 备用方案：句子分片（`--sentence` 参数）
